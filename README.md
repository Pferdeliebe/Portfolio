
# Stock Price Analysis with Pandas and Matplotlib

## Project Description

This project simulates daily stock price changes, calculates key statistical metrics, and visualizes the stock price development using a line chart.

The goal is to demonstrate basic data analysis and visualization skills using Python.

---

## Features

- Simulation of daily stock price changes
- Calculation of:
  - Maximum value
  - Minimum value
  - Arithmetic mean
  - Standard deviation
- Visualization of stock price development using Matplotlib
- Storage of values in a DataFrame using Pandas

---

## Technologies Used

- Python
- Pandas
- Matplotlib
- Random module

---

## How It Works

1. Random daily price changes are generated
2. A time series is calculated based on these changes
3. Statistical values are computed
4. Data is stored in a Pandas DataFrame
5. A line chart visualizes the stock price development
